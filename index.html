Import from Google Sheets
Paste your CSV data exported from Google Sheets to bulk-import into Firebase
How to export from Google Sheets
1. In your Google Sheet, go to File → Download → CSV (.csv)
2. Open the downloaded file in a text editor and copy all the content
3. Paste it below. Your CSV must have these columns (in any order):
date, account, category, item, amount, toFrom

Date format accepted: DD/MM/YYYY, YYYY-MM-DD, MM/DD/YYYY
Amount: positive = income, negative = expense
